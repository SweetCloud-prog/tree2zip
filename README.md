# 📂 tree2zip
> Allows you to create a zip file countaining all of your future app structure

![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![GitHub stars](https://img.shields.io/github/stars/SweetCloud_prog/SweetCloud_prog)


**tree2zip** is a lightweight Python tool app allowing you to generate instantly a complex file structure ( `.zip` format) from a simple copy-paste or from a `.txt` file. The endless manual structure creation is over !

---

## ✨ Features
- 📝 **Flexible Input**: Copy your text or import a `.txt` file.
- 📂 **Native File Explorer**: Select your files using Windows/macOS/Linux File Explorer.
- ⚡ **Instant Generation**: Creates a ready-to-use `.zip` file.
- 🛠️ **Smart Parsing**: Supports indentation (spaces or tabs).

```bash
# Clone the repository

# Install dependencies
